## üíª Coding Standards: "Elegant Code"
Elegant code is the baseline for this project. It is defined by:
1. **Readable at a Glance:** The structure should be graspable in under 30 seconds of skimming.
2. **Self-Documenting:** Variable and function names should carry the weight. Comments should explain **why** something is done, not just **what**.
3. **Minimally Complex:** Use the simplest construct that solves the problem. Avoid "clever" one-liners or over-engineering.
4. **Spatially Organized:** Cluster related logic together to support 3D pattern recognition.
5. **Defensively Modest:** Handle edge cases and validate inputs without excessive ceremony or paranoia.
