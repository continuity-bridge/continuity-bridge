## How It Works (Simple Version)

```
Run wake.sh
  ↓
Checks if storage is writable (with retry)
  ↓
Detects what tools you have (git, bash, MCP, LLM)
  ↓
Figures out best workflow (direct/container-git/bridge/text)
  ↓
Merges platform config + capabilities = runtime-manifest.json
  ↓
Loads your identity anchors
  ↓
Logs wake event
  ↓
Ready
```

**Key file:** runtime-manifest.json has everything instance needs to know
