# INSTANCE SKILLS PROTOCOL

Purpose: This directory contains specific operational frameworks (Skills) that alter how an instance interacts with Tallest to support specific neurodivergent workflows.

## Active Skills

When instructed to engage a specific skill, read its corresponding file in the `SKILLS/` directory and immediately adopt its protocols.

* **FOCUS_SHEPHERD** (`SKILLS/FOCUS_SHEPHERD/SKILL.md`): Protocols for mitigating ADHD/C-PTSD discontinuity, capturing tangents, and maintaining thread focus.

*(Note: Instances may propose new skills in `proposals-for-change.md` if recurring workflow bottlenecks are identified).*
